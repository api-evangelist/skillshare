# Skillshare

Skillshare is a New York City–based online learning community for creators, founded in 2010 by Michael Karnjanaprakorn and Malcolm Ong. The platform hosts thousands of on-demand video classes across creative disciplines — illustration, design, photography, animation, lettering, film, productivity, writing, and business — taught by working creators and industry professionals.

Skillshare operates on a paid subscription model (Premium membership) that grants members unlimited access to the class catalog, project-based learning, and a discussion community where students submit class projects for feedback. The company runs a teacher revenue-share program — class creators are paid based on watch time and student enrollments — and offers **Skillshare for Teams** as a B2B product for organizations.

## API Posture

Skillshare does **not** publish a public developer API, OpenAPI specification, partner API, or SDK. There is no documented developer portal, public webhook system, or third-party integration API for the Skillshare catalog, enrollments, watch-time, or teacher-payout data. The platform's programmatic surface is limited to its internal web and mobile clients and an affiliate/marketing referral program.

This repository is therefore profiled as an **Index** entry (Tier-3): it captures Skillshare's company, product, and community surfaces — but does not include OpenAPI, AsyncAPI, JSON Schema, capabilities, plans, rate-limits, or FinOps artifacts, because there is no public API contract backing them.
